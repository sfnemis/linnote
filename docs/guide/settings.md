Access settings via `Ctrl+,` or `/settings`.

**Light themes:**

- **Font Family** - System or custom font
- **Font Size** - 10-24px

### Window

- **Always on Top** - Keep window above others
- **Start Minimized** - Launch to system tray
- **Remember Position** - Save window location

## Behavior

### Auto-save

- **Enabled** - Save automatically (default)
- **Interval** - Save every X seconds

### Startup

- **Launch at Login** - Start with system
- **Start Mode** - Normal, minimized, or tray

### Updates

- **Check for Updates** - Automatic update check
- **Update Channel** - Stable or beta

## Global Hotkey

Set a keyboard shortcut to show/hide LinNote from anywhere.

### Configure

1. **Settings** → **Shortcuts**
2. Click **Global Hotkey** field
3. Press your desired key combination
4. Example: `Super+N`, `Ctrl+Alt+N`

### Notes

- Requires KDE Plasma for global shortcuts
- May need to configure in System Settings → Shortcuts
