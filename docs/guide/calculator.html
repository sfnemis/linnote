Calculator ​

LinNote's calculator evaluates expressions in real-time.

Activate:

calc

Basic Usage ​

Type any expression. Add = at the end to see the result:

2 + 2 =
// Result: 4

100 / 4 =
// Result: 25

// Result: 500

// Result: 50

// Result: 450
