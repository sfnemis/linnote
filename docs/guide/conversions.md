# Conversions

LinNote can convert currencies and units in real-time.

## Currency Conversion

### Syntax

```
100 USD to EUR =
50 EUR in TRY =
1000 TRY to USD =
```

### Supported Currencies

**Major:**
USD, EUR, GBP, JPY, CHF, CAD, AUD, NZD

**Other:**
TRY, RUB, INR, CNY, KRW, BRL, MXN, PLN, SEK, NOK, DKK, CZK, HUF, RON, BGN, HRK, ISK, THB, SGD, HKD, MYR, PHP, IDR, VND, ZAR, AED, SAR, ILS, EGP

**Crypto:**
BTC, ETH

### Examples

```
100 USD to EUR =
// Result: ~92.50 EUR

1 BTC to USD =
// Result: ~43,000 USD

500 EUR to TRY =
// Result: ~17,500 TRY
```

> **Note:** Exchange rates update every hour from online sources.

## Unit Conversion

### Length

```
10 km to miles =
5 feet to cm =
100 m to yards =
```

**Supported:** km, m, cm, mm, miles, yards, feet, inches

### Weight

```
5 kg to lbs =
100 lbs to kg =
200 g to oz =
```

**Supported:** kg, g, mg, lbs, oz, tons

### Temperature

```
100 C to F =
72 F to C =
300 K to C =
```

**Supported:** C (Celsius), F (Fahrenheit), K (Kelvin)

### Volume

```
1 gallon to liters =
500 ml to cups =
```

**Supported:** L, mL, gallons, quarts, pints, cups, fl oz

### Data

```
1 GB to MB =
500 MB to GB =
1 TB to GB =
```

**Supported:** B, KB, MB, GB, TB, PB

### Time

```
2 hours to minutes =
3600 seconds to hours =
```

**Supported:** ms, seconds, minutes, hours, days, weeks
