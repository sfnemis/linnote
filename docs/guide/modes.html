Supported Languages ‚Äã

C, C++, Python, JavaScript, TypeScript, Rust, Go, Java, Kotlin, Swift, Ruby, PHP, HTML, CSS, JSON, YAML, SQL, Bash

Usage Example ‚Äã

python
def fibonacci(n):
    a, b = 0, 1
    result = []
    while a < n:
        result.append(a)
        a, b = b, a + b
    return result

print(fibonacci(100))
