# Calculator

The built-in calculator evaluates expressions in real-time.

## Basic Usage

Type any expression ending with `=`:

```
2 + 2 =
// Result: 4

100 / 4 =
// Result: 25
```

## Operators

| Operator | Description |
|----------|-------------|
| `+` | Addition |
| `-` | Subtraction |
| `*` | Multiplication |
| `/` | Division |
| `^` | Power |
| `%` | Modulo |
| `()` | Parentheses |

## Percentages

```
100 + 18% =
// Result: 118

500 - 20% =
// Result: 400

what is 15% of 200 =
// Result: 30
```

## Variables

Store values for reuse:

```
price = 100
tax = 18
total = price + (price * tax / 100) =
// Result: 118
```

## Functions

| Function | Description |
|----------|-------------|
| `sqrt(x)` | Square root |
| `abs(x)` | Absolute value |
| `sin(x)` | Sine (radians) |
| `cos(x)` | Cosine |
| `tan(x)` | Tangent |
| `log(x)` | Natural log |
| `log10(x)` | Log base 10 |
| `exp(x)` | e^x |
| `floor(x)` | Round down |
| `ceil(x)` | Round up |
| `round(x)` | Round nearest |
| `min(a,b)` | Minimum |
| `max(a,b)` | Maximum |

## Constants

| Constant | Value |
|----------|-------|
| `pi` | 3.14159... |
| `e` | 2.71828... |

## Examples

```
sqrt(144) =
// Result: 12

sin(pi/2) =
// Result: 1

2^10 =
// Result: 1024

max(5, 10, 3) =
// Result: 10
```
