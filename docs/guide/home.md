# LinNote Documentation

Welcome to the LinNote documentation! LinNote is a fast, keyboard-driven scratchpad for Linux built with Qt6.

## Quick Install

```bash
curl -fsSL https://linnote.app/install.sh | bash
```

## Features

- **6 Note Modes** - Plain Text, Checklist, Code, Math, Timer, Markdown
- **Smart Calculator** - Type math expressions with `=` to evaluate
- **Currency Conversion** - Real-time rates for 30+ currencies
- **Unit Conversion** - Length, weight, temperature, and more
- **Built-in Timers** - Stopwatch, countdown, Pomodoro
- **OCR Capture** - Extract text from screen
- **Note Encryption** - Protect with master password
- **14 Themes** - Light and dark options

## Getting Started

1. Install LinNote using the command above
2. Launch from your application menu or run `linnote`
3. Start typing! Use `/` to access slash commands

## Navigation

Use the sidebar to explore different features:

- **Note Modes** - Learn about each mode
- **Slash Commands** - Quick actions with `/`
- **Calculator** - Math, variables, functions
- **Conversions** - Currency and units
- **Timer** - Pomodoro and stopwatch
