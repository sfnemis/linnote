Global Hotkey ​

Set a keyboard shortcut to show/hide LinNote from anywhere.

Configure ​

  1. Settings → Shortcuts
  2. Click Global Hotkey field
  3. Press your desired key combination
  4. Example: Super+N, Ctrl+Alt+N

Notes ​

  • Requires KDE Plasma for global shortcuts
  • May need to configure in System Settings → Shortcuts

⚠️ IMPORTANT: After changing shortcuts, you must restart LinNote for changes to take effect.
