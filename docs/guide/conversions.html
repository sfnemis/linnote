Conversions ​

LinNote can convert currencies and units in real-time.

Currency Conversion ​

Syntax ​

100 USD to EUR =
50 EUR in TRY =
1000 TRY to USD =

Supported Currencies ​

Major: USD, EUR, GBP, JPY, CHF, CAD, AUD, NZD

Other: TRY, RUB, INR, CNY, KRW, BRL, MXN, PLN, SEK, NOK, DKK, CZK, HUF, RON, BGN, HRK, ISK, THB, SGD, HKD, MYR, PHP, IDR, VND, ZAR, AED, SAR, ILS, EGP

Crypto: BTC, ETH

Examples ​

100 USD to EUR =
// Result: ~92.50 EUR

1 BTC to USD =
// Result: ~43,000 USD

500 EUR to TRY =
// Result: ~17,500 TRY

Note: Exchange rates update every hour from online sources.
